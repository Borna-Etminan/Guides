# 📚 Guides Repository

This repository is a collection of structured guides based on what I learn throughout my development journey.

Whenever I study a technical topic, I rewrite it in a clear, simple, and organized way — both to reinforce my own understanding and to create clean reference materials for others.

The goal of this repository is not just documentation, but clarity and mastery and (Fast learning with conciseness) through structured learning.

---

## 📂 Available Guides

### 🐧 Linux Terminal Guide  
📁 [`Linux-guide`](https://github.com/Borna-Etminan/Guides/tree/main/Linux%20Guide)  
A practical introduction to Linux terminal commands, covering file management, processes, permissions, environment variables, package managers, and essential command-line tools.

---

### 🌿 Git Guide  
📁 [`git-guide`](https://github.com/Borna-Etminan/Guides/tree/main/Git%20Guide)  
A structured Git reference covering version control fundamentals, branching, merging, collaboration workflows, and real-world command usage.

---

## 🧠 Philosophy

- Learn deeply  
- Rewrite clearly  
- Structure knowledge  
- Practice consistently  

Each guide is written in a minimal, practical, and example-driven style.

---

## 🚀 How to Navigate

1. Open the folder of the guide you want.
2. Read the README inside.
3. Download the PDF (if available).
4. Practice the commands locally.
